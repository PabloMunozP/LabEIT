Recuerde el formato de este archivo y su extension (.csv o .xlsx). ¿No conoce el formato? Haga click aquí.

Formato CSV

Para obtener de forma más sencilla la lista de los alumnos a ingresar se sugiere exportar los datos desde Canvas. Para esto deberá dirigirse al apartado 'Calificaciones' y exportar en formato CSV.
Una vez que haya descargado la lista de alumnos deberá modificar algunas de sus columnas, ya que el sistema Canvas da más información de la necesitada para el registro de usuarios.
Para esto será necesario solo dejar las columnas con los nombres 'Student, SIS User ID, SIS Login ID' que representan los campos de nombre, RUT y correo respectivamente de cada alumno.

Formato XLSX

Si desea ingresar los valores de manera mas controlada en una hoja de cálculo, puede probar con el formato xlsx haciendo click aquí para descargar una plantilla y posteriormente subirla con los datos cargados.

Considere que el tamaño máximo permitido para el archivo es de 5 Mb.
